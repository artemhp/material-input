Validation example

Usage with Bootstrap 4

  1. Just add class material-form-group to .form-group or nearest container.
  2. label tag should be after input.
  3. placeholder in input should not be empty but can consist anything.
                
<div class="form-group material-form-group">
    <input type="email" placeholder="M" class="form-control" id="example3">
    <label for="example3">Email address</label>
</div>

With wrapped label (no id):

                
<label class="form-group material-form-group">
    <input type="email" placeholder="M" class="form-control">
    <label>Email address</label>
</label>
